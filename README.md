# T Kushal Raj — Portfolio

Personal portfolio site built with **React** and **Vite**. Highlights work as a React Native and game developer based in Bengaluru, India.

## Tech stack

- React 18
- Vite 8

## Run locally

```bash
npm install
npm run dev
```

Open the URL Vite prints (usually `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```

`preview` serves the production build locally for a quick check before deploy.
